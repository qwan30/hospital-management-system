# Admin Monitoring Page Overrides

> PROJECT: Clinical Atelier
> Page Type: Administrative monitoring dashboard

These rules override the master design system for the `/admin-monitoring` route.

## Layout

- Lead with health and alert context before audit detail.
- Pair a compact health snapshot with a longer audit stream.

## Required Sections

1. Admin KPI row.
2. Runtime health snapshot.
3. Audit activity stream.

## Tone

- Calm control room, not an error console.
- Use red only for unhealthy states or elevated alerts.
