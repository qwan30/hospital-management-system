# Patient Records Management Page Overrides

> PROJECT: Clinical Atelier
> Page Type: Internal records search

These rules override the master design system for the `/patient-records-management` route.

## Layout

- Search and result selection must appear before deep chart detail.
- Use a dual-pane layout: matched patients on one side, selected chart snapshot on the other.

## Required Sections

1. Search input with immediate filtering.
2. Result list of matched patients.
3. Selected patient snapshot.
4. Appointment history timeline.

## Tone

- Dense but clinical.
- Emphasize identity, chronology, and diagnosis clarity.
