# Website Section Editor Page Overrides

> PROJECT: Clinical Atelier
> Page Type: Split-pane editor

These rules override the master design system for the `/cms/sections/[sectionId]` route.

## Layout

- Use a split-pane editor: main editing canvas plus metadata/preview sidebar.
- Keep save and navigation actions visible without requiring deep scrolling.

## Required Sections

1. Editor header with save/back actions.
2. Content form fields.
3. CTA and media controls.
4. Sidebar metadata and preview.

## Tone

- Editing surface should feel focused and quiet.
- Avoid dashboard clutter inside the editor pane.
